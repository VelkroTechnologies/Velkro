# Velkro Engine
Velkro Engine's source code is free to view and modify to your needs, however you are not permited to use
Velkro Engine's source code commercially.

Velkro Engine supports linux, and windows. It has plans to support MacOS in the future, however probably not soon.

**Dependencies**
- Premake

**How do you build Velkro Engine?**

First, clone Velkro Engine and it's submodules recursively.

``git clone https://github.com/VelkroTechnologies/Velkro/ --recursive``

Then using premake run premake5 and generate project files for your environment.
e.g. Visual Studio 2022 

``premake5 vs2022`` 

or Make 

``premake5 gmake2``

Now, you can build Velkro Engine with your environment, and even contribute!
